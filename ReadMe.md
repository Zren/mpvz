# mpvz

Qt5/QML based GUI for libmpv. Based on the Tethys theme for [Bomi](http://bomi-player.github.io/).

`mpvz https://www.youtube.com/watch?v=j_aDGFXNfyw`

## Compile and Install

```
mkdir -p build
cd build
qmake ..
make
sudo make install
```
